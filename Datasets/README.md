# Information on the Datasets
## statewide_reservoir_storage_2022_2026.csv
- STATION_ID: where SVW is Statewide Storage Estimate
- DURATION: Daily or Monthly
- SENSOR_NUMBER: Reservoir Storage
- SENSOR_TYPE: Storage
- DATE_TIME: YYYY-MM-DD TTTT
- OBS_DATE: Date observed
- VALUE: Reservoir level
- DATA_FLAG: Data validation
- UNIT: Acre-Foot

## drought_010122_032626.csv
- MapDate: YYYYMMDD
- None: No Drought 
- D0: Abnormally Dry
- D1: Moderate Drought
- D2: Severe Drought
- D3 Extreme Drought
- D4: Exceptional Drought
- ValidStart: Start of data collection
- ValidEnd: End of data collection
